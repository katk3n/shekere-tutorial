## 目標
これまで学習した知識を統合し、オリジナルのシェーダーアートを作成する。

## 課題内容
Chapter 1で学習した以下の要素を組み合わせて、オリジナル作品を作成してください：

### 使用可能な要素
- **色の指定**: RGB値による色表現
- **UV座標**: `NormalizedCoords()`による位置情報
- **基本関数**: `length()`, `abs()`, `sin()`, `cos()`など
- **条件分岐**: `if`文による場合分け

### 作品のテーマ（いずれか選択）

#### テーマA: 幾何学パターン
以下の要素を含む幾何学的なパターンを作成：
- 複数の色を使用
- 規則的な形状（円、四角、三角など）
- 対称性のあるデザイン

例：同心円、格子模様、放射状パターンなど

#### テーマB: 自然モチーフ
自然現象をイメージしたパターンを作成：
- 夕焼け、虹、海、山などをイメージ
- グラデーションを効果的に使用
- 有機的な形状

例：夕焼けグラデーション、波模様、雲のパターンなど

#### テーマC: 自由創作
完全に自由なテーマで作品を作成：
- あなたの好きなもの、感情、抽象的概念など
- 色彩とパターンで表現
- 独創性を重視

### 実装のヒント

#### 条件分岐の使用例
```wgsl
@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let uv = NormalizedCoords(in.position.xy);
    
    var color = vec3(0.0); // 初期色は黒
    
    // 中心からの距離で色を変える
    let distance = length(uv);
    if distance < 0.3 {
        color = vec3(1.0, 0.0, 0.0); // 赤い円
    } else if distance < 0.6 {
        color = vec3(0.0, 1.0, 0.0); // 緑のリング
    } else {
        color = vec3(0.0, 0.0, 1.0); // 青い背景
    }
    
    return vec4(ToLinearRgb(color), 1.0);
}
```

#### 数学関数の活用例
```wgsl
// 波状パターン
```
