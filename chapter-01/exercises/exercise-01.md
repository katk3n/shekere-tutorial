# 演習課題 1: 好きな色の作成

## 目標
RGB値を理解し、自分の好きな色を作成できるようになる。

## 課題内容
`examples/01-solid-color`をベースにして、以下の色を作成してください：

### 1-1. 基本色の作成
以下の色を順番に作成し、それぞれの結果を確認してください：

1. **純粋な緑色**
2. **純粋な青色**
3. **黄色**（赤と緑の混合）
4. **マゼンタ**（赤と青の混合）
5. **シアン**（緑と青の混合）

### 1-2. 中間色の作成
以下の中間色を作成してください：

1. **オレンジ色**
   - ヒント: 赤=1.0, 緑=0.5程度, 青=0.0
2. **紫色**
   - ヒント: 赤=0.5程度, 緑=0.0, 青=1.0
3. **ピンク色**
   - ヒント: 高い赤成分と適度な緑と青成分

### 1-3. 自由課題
あなたの好きな色を3つ作成してください。それぞれの色について：
- RGB値をメモしてください
- なぜその色を選んだか理由を書いてください

## 実践方法
以下の形式でコードを書いて確認してください：

```wgsl
// 1-1-1. 純粋な緑色
let color = vec3(0.0, 1.0, 0.0);

// 1-2-1. オレンジ色
let color = vec3(1.0, 0.5, 0.0);

// 1-3-1. 私の好きな色1: [色の名前]
// 理由: [選んだ理由]
let color = vec3(?, ?, ?);
```

## 学習のポイント
- RGB各成分は0.0〜1.0の範囲で指定します
- 値を変更したらすぐに結果を確認する習慣をつけましょう
- Hot Reload機能を活用して、リアルタイムで変化を観察しましょう
