# 演習 1: 基本の円を描く

- **length()**: 中心からの距離計算（距離関数の基礎）
- **条件分岐**: `if`文による領域分割
- **天体の色彩設計**: 太陽と宇宙背景の色表現

## 🎨 作品コンセプト
**統合作品**: 「太陽の輪」- Chapter 1技術を統合した太陽系アート
**第1段階**: 基本の太陽円 - 単純な円形で太陽の基本形状を確立

## 📝 段階的実装手順

以下の6つのステップを順番に実装することで、太陽の基本形を段階的に構築します。

### Step 1: 環境確認（基本設定）

**目標**: shekere環境の動作確認と基本的な色表示

**新規追加コード**:
```wgsl
@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    // 単色で環境確認
    let color = vec3(0.8, 0.4, 0.0); // 温かいオレンジ色
    return vec4(ToLinearRgb(color), 1.0);
}
```

**解説**: 最もシンプルな色表示で環境が正常に動作することを確認します。

**確認方法**: 画面全体がオレンジ色で表示されることを確認してください。

---

### Step 2: UV座標系の導入

**目標**: `NormalizedCoords()`を使用してUV座標系を確立

**新規追加コード**:
```wgsl
// 新規追加: UV座標系の導入
let uv = NormalizedCoords(in.position.xy);

// UV座標をデバッグ表示（X座標を赤成分に）
let color = vec3((uv.x + 1.0) * 0.5, 0.3, 0.1);
```

**解説**: `NormalizedCoords()`でUV座標(-1〜1の範囲)を取得し、座標値を色で表示します。

**確認方法**: 左右方向に色が変化することを確認してください。

---

### Step 3: 距離計算の追加

**目標**: `length()`関数で中心からの距離を計算

**新規追加コード**:
```wgsl
// 新規追加: 中心からの距離を計算
let distance = length(uv);

// 距離を色で可視化
let color = vec3(distance * 0.8, distance * 0.4, 0.1);
```

**解説**: `length(uv)`で中心(0,0)からの距離を計算し、距離に応じて色を変化させます。

**確認方法**: 中心が暗く、外側に向かって明るくなるグラデーションが表示されることを確認してください。

---

### Step 4: 背景色の設定

**目標**: 宇宙をイメージした背景色を設定

**新規追加コード**:
```wgsl
// 新規追加: 宇宙背景色の設定
var color = vec3(0.0, 0.0, 0.2); // 深い宇宙色
```

**解説**: 変数`color`を背景色で初期化し、後で太陽領域を上書きする準備をします。

**確認方法**: 画面全体が深い青色（宇宙色）で表示されることを確認してください。

---

### Step 5: 条件分岐による太陽領域の作成

**目標**: `if`文を使って距離による領域分割を実装

**新規追加コード**:
```wgsl
// 新規追加: 太陽領域の条件分岐
if distance < 0.7 {
    color = vec3(1.0, 0.6, 0.0); // 温かいオレンジ色の太陽
}
```

**解説**: 中心から半径0.7の範囲を太陽領域として、温かい色で塗りつぶします。

**確認方法**: 中央に大きなオレンジ色の円（太陽）が描画されることを確認してください。

---

### Step 6: 最終調整

**目標**: 色彩を微調整して第1段階を完成

**最終コード**:
```wgsl
@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let uv = NormalizedCoords(in.position.xy);
    let distance = length(uv);
    var color = vec3(0.0, 0.0, 0.2); // 背景: 深い宇宙色
    
    // 太陽の本体（半径0.7の温かい光）
    if distance < 0.7 {
        color = vec3(1.0, 0.6, 0.0); // 温かいオレンジ色
    }
    
    return vec4(ToLinearRgb(color), 1.0);
}
```

**確認方法**: 深い宇宙の背景に美しいオレンジ色の太陽が描画されることを確認してください。

## 🔧 技術解説

### UV座標系
```wgsl
let uv = NormalizedCoords(in.position.xy);
```
- 画面を-1〜1の範囲に正規化
- 中心が(0,0)、右上が(1,1)、左下が(-1,-1)
- 画面の縦横比は自動調整される

### 距離関数
```wgsl
let distance = length(uv);
```
- `length()`: ベクトルの大きさ（原点からの距離）
- 中心=0.0、画面端≈1.4の範囲
- 円形パターンの基礎となる重要な関数

### 条件分岐
```wgsl
if distance < 0.7 { ... }
```
- 距離による領域分割
- 円の内側と外側で異なる色を適用
- より複雑なパターンの基礎
