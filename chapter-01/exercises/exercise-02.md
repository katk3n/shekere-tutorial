# 演習課題 2: グラデーションの変化

## 目標
UV座標と色の関係を理解し、様々なグラデーション効果を作成できるようになる。

## 課題内容
`examples/02-gradient`をベースにして、以下のグラデーションを作成してください：

### 2-1. 方向別グラデーション
以下の方向のグラデーションを作成してください：

1. **横方向のグラデーション**
   - 左端: 黒色 → 右端: 白色
   - ヒント: Y座標は使わずX座標のみ使用

2. **縦方向のグラデーション**
   - 下端: 黒色 → 上端: 白色
   - ヒント: X座標は使わずY座標のみ使用

3. **対角線グラデーション**
   - 左下: 黒色 → 右上: 白色
   - ヒント: X座標とY座標の平均値を使用

### 2-2. カラーグラデーション
以下の色のグラデーションを作成してください：

1. **赤から青へのグラデーション**
   - 左端: 純粋な赤 → 右端: 純粋な青

2. **虹色グラデーション**
   - 左端: 赤 → 中央: 緑 → 右端: 青
   - ヒント: 複数の条件分岐や数学関数を使う

### 2-3. 円形グラデーション
以下の円形パターンを作成してください：

1. **中心からの距離グラデーション**
   - 中心: 白色 → 外側: 黒色
   - ヒント: `length(uv)`で中心からの距離を計算

```wgsl
let distance = length(uv);
let intensity = 1.0 - distance; // 距離が大きいほど暗く
```

## 学習のポイント
- UV座標は-1.0〜1.0の範囲です
- `length()`関数で距離を計算できます
- `abs()`関数で絶対値を取れます
- 数学関数（sin, cos, など）も使用可能です
