# 演習 2: 同心円パターンを作る

## 目標
前回作成した単色の円を発展させて、3層の同心円（カラフルなリング）を作成する。

## 学習内容
- **複数の色**: RGB値による様々な色の指定
- **領域分け**: 複数の`if`文による条件分岐
- **段階的な構築**: 前のステップの成果を活用

## 課題の流れ
この演習では、「太陽の輪」作品の第2ステップとして、赤・黄・オレンジの3層の同心円を作成します。

## 実装手順

### ステップ1: 前回のコードから開始
まず、前回作成した基本的な円のコードを確認しましょう。

```wgsl
@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let uv = NormalizedCoords(in.position.xy);
    let distance = length(uv);
    
    var color = vec3(0.0);
    
    if distance < 0.5 {
        color = vec3(1.0, 0.0, 0.0); // 赤色
    }
    
    return vec4(ToLinearRgb(color), 1.0);
}
```

### ステップ2: 3つの領域に分ける
距離に応じて3つの異なる色の領域を作成します。

```wgsl
@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let uv = NormalizedCoords(in.position.xy);
    let distance = length(uv);
    
    var color = vec3(0.0);
    
    if distance < 0.3 {
        // 内側の円: 温かい赤
        color = vec3(1.0, 0.3, 0.1);
    } else if distance < 0.6 {
        // 中間のリング: 黄色
        color = vec3(1.0, 0.8, 0.0);
    } else if distance < 0.9 {
        // 外側のリング: オレンジ
        color = vec3(1.0, 0.5, 0.0);
    }
    // distance >= 0.9 の場合は黒のまま（背景）
    
    return vec4(ToLinearRgb(color), 1.0);
}
```

### ステップ3: 色を調整する
太陽のイメージに合うように色を微調整してみましょう。

## 完成目標
以下の3層の同心円が描画されることを確認してください：
- 内側（半径0.3まで）: 温かい赤色
- 中間（半径0.3〜0.6）: 明るい黄色
- 外側（半径0.6〜0.9）: 鮮やかなオレンジ色
- 背景（半径0.9以上）: 黒色

## 次のステップへ
この同心円ができたら、次の演習で放射状のライン（光線）を追加していきます。

## 学習のポイント
- `else if`を使って複数の条件を順番にチェックできます
- RGB値の組み合わせで様々な色を作れます
- 距離による領域分けで複雑なパターンが作れます
- 各領域の境界値（0.3, 0.6, 0.9）を変更してサイズを調整できます

## 発展課題
余裕があれば以下にも挑戦してみてください：
1. リングのサイズを変更してみる
2. 異なる色の組み合わせを試してみる
3. より多くのリング（4層、5層）を作ってみる
4. 背景色を黒以外にしてみる

## RGB色の参考
```wgsl

vec3(1.0, 0.0, 0.0)  // 赤
vec3(0.0, 1.0, 0.0)  // 緑

// 太陽系の色

vec3(1.0, 0.5, 0.0)  // 鮮やかなオレンジ
```
