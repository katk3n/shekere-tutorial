## 📝 段階的実装手順

以下の6つのステップで前ステップのコードを継承・拡張し、同心円構造を構築します。

### Step 1: 前ステップコードの継承確認

**目標**: 前ステップの基本太陽コードが正常に動作することを確認

**継承コード**:
```wgsl
@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let uv = NormalizedCoords(in.position.xy);
    let distance = length(uv);
    var color = vec3(0.0, 0.0, 0.2); // 背景: 深い宇宙色
    
    // 太陽の本体（前ステップから継承）
    if distance < 0.7 {
        color = vec3(1.0, 0.6, 0.0); // 温かいオレンジ色
    }
    
    return vec4(ToLinearRgb(color), 1.0);
}
```

**解説**: 前ステップで作成した基本太陽が正常に表示されることを確認します。

**確認方法**: オレンジ色の太陽が宇宙背景に表示されることを確認してください。

---

### Step 2: 太陽領域の細分化開始

**目標**: 単一の太陽領域を2つに分割する準備

**変更コード**:
```wgsl
// 【変更】: 太陽領域を2段階に分割
if distance < 0.35 {
    color = vec3(1.0, 0.8, 0.5); // 内側: より高温色
} else if distance < 0.7 {
    color = vec3(1.0, 0.6, 0.0); // 外側: 元の太陽色（継承）
}
```

**解説**: else if文を導入し、太陽を内側と外側の2層に分割します。

**確認方法**: 太陽の中心がより明るい色になることを確認してください。

---

### Step 3: 3層構造の完成

**目標**: 太陽の温度勾配を表現する3層同心円を実装

**新規追加コード**:
```wgsl
// 【新規追加】: 3層による太陽の温度勾配表現
if distance < 0.25 {
    color = vec3(1.0, 0.9, 0.7); // 核心: 最高温の白熱色
} else if distance < 0.45 {
    color = vec3(1.0, 0.7, 0.2); // 中間層: 黄橙色
} else if distance < 0.7 {
    color = vec3(1.0, 0.6, 0.0); // 外層: 温かいオレンジ色
}
```

**解説**: 3つの距離条件で太陽内部の温度変化を表現します。

**確認方法**: 中心から外側に向かって白→黄→オレンジと変化することを確認してください。

---

### Step 4: 色彩の科学的調整

**目標**: より科学的な太陽の色温度を表現

**色彩調整**:
```wgsl
if distance < 0.25 {
    color = vec3(1.0, 0.95, 0.8); // 核心: 6000K相当の白色
} else if distance < 0.45 {
    color = vec3(1.0, 0.8, 0.3);  // 中間層: 4000K相当
} else if distance < 0.7 {
    color = vec3(1.0, 0.6, 0.1);  // 外層: 3000K相当
}
```

**解説**: 太陽の実際の温度分布に近い色彩に調整します。

**確認方法**: より自然で美しい太陽の色温度グラデーションを確認してください。

---

### Step 5: エッジの微調整

**目標**: 各層の境界を最適化

**境界調整**:
```wgsl
if distance < 0.25 {
    color = vec3(1.0, 0.9, 0.7);  // 太陽の核心
} else if distance < 0.45 {
    color = vec3(1.0, 0.7, 0.2);  // 太陽の中間層
} else if distance < 0.7 {
    color = vec3(1.0, 0.6, 0.0);  // 太陽の外層（前ステップ継承領域）
}
```

**解説**: 各層のサイズバランスを調整して視覚的に最適化します。

**確認方法**: 各層が適切な比率で表示されることを確認してください。

---

### Step 6: 最終コード完成

**目標**: 第2段階の完成コードを確定

**最終コード**:
```wgsl
@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    // 【前ステップから継承】: 基本設定
    let uv = NormalizedCoords(in.position.xy);
    let distance = length(uv);
    var color = vec3(0.0, 0.0, 0.2); // 背景: 深い宇宙色
    
    // 【新規追加】: 3層同心円による太陽構造
    if distance < 0.25 {
        color = vec3(1.0, 0.9, 0.7); // 太陽の核心
    } else if distance < 0.45 {
        color = vec3(1.0, 0.7, 0.2); // 太陽の中間層
    } else if distance < 0.7 {
        color = vec3(1.0, 0.6, 0.0); // 太陽の外層
    }
    
    return vec4(ToLinearRgb(color), 1.0);
}
```

**確認方法**: 美しい3層太陽が完成していることを確認してください。
