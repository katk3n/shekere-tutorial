# Chapter 1 解答例

このディレクトリには、Chapter 1の演習課題の解答例が含まれています。

## 解答例の構成

### solution-01: 好きな色の作成
- **内容**: RGB値による様々な色の作成例
- **学習ポイント**: 基本色、中間色、パステルカラーの理解
- **特徴**: コメントアウトで複数の色の例を提示

### solution-02: グラデーションの変化
- **内容**: 方向別グラデーション、カラーグラデーション、円形グラデーション
- **学習ポイント**: UV座標の活用、数学関数の使用
- **特徴**: 発展的なパターン（波状、チェッカー）も含む

### solution-03: 創作課題
- **作品名**: 「太陽の輪」
- **テーマ**: 幾何学パターン
- **技術**: 同心円、放射状パターン、条件分岐、色のブレンド

## 使用方法

各解答例を実行するには：

```bash
cd solution-01
shekere config.toml
```

## 学習の進め方

1. **まず自分で挑戦**: 解答例を見る前に、まず自分で演習課題に取り組んでください
2. **比較検討**: 自分の解答と解答例を比較し、異なるアプローチを学びましょう
3. **改良実験**: 解答例をベースに、さらなる改良や変化を試してみましょう
4. **理解確認**: コメントを読んで、各技術の仕組みを理解しましょう

## 重要な学習ポイント

### 使用されている技術
- `length()`: 中心からの距離計算
- `atan2()`: 角度計算
- `mix()`: 色のブレンド
- `smoothstep()`: 滑らかな変化
- `clamp()`: 値の範囲制限
- `step()`: 閾値による二値化
- `fract()`: 小数部分取得（繰り返しパターン）

### デザインの考え方
- **レイヤー構造**: 複数の要素を重ね合わせる
- **領域分割**: 条件分岐で画面を領域に分ける
- **グラデーション**: 急激な変化を避けて自然な見た目にする
- **対称性**: 規則的なパターンで美しさを演出

## 次のステップ

解答例を理解したら、以下に挑戦してみましょう：

1. **色の変更**: 異なる色合いで同じパターンを試す
2. **パラメータ調整**: 数値を変更して効果の変化を観察
3. **組み合わせ**: 複数の解答例の技術を組み合わせる
4. **オリジナル作品**: 学んだ技術で完全にオリジナルな作品を作成

頑張って学習を続けてください！
