# Chapter 2 演習課題 解答例

## 概要
このフォルダには、Chapter 2の演習課題の解答例が含まれています。これらの解答例は、課題の一つの解決方法を示すものであり、唯一の正解ではありません。

## 解答例の内容

### Solution 1: 基本的な時間アニメーションの応用
**対応課題**: `exercises/exercise-01.md`

**実装内容**:
- 呼吸効果: 画面全体が呼吸するような明度変化
- 時間の2乗を利用した加速アニメーション
- 複数の三角関数を組み合わせた複雑な色変化

**学習ポイント**:
- Time.durationの様々な活用方法
- sin/cos関数の組み合わせ
- 色の範囲変換テクニック

---

### Solution 2: sin/cosを使った様々なパターン
**対応課題**: `exercises/exercise-02.md`

**実装内容**:
- 同心円パターン: 中心から広がる円形波動
- 螺旋パターン: 距離と角度を組み合わせた螺旋効果
- 複合波動: 複数の波動を重ね合わせたモアレ風パターン

**学習ポイント**:
- length()とatan2()の活用
- 複数の波動の重ね合わせ
- 幾何学的パターンの生成

---

### Solution 3: 創作課題の例
**対応課題**: `exercises/exercise-03.md`

**実装内容**:
- テーマ: 「海の波」の表現
- 複数の波動を組み合わせた海面の動き
- 時間による色の変化で一日の移り変わりを表現
- 距離による波の減衰効果

**学習ポイント**:
- 自然現象の抽象化
- 複数の技術要素の統合
- 創造的な表現の実現

## 解答例の活用方法

### 1. 自分の解答との比較
自分の実装と比較して、異なるアプローチや技術を学んでください。

### 2. 理解の深化
コメントを読んで、実装の意図や技術的なポイントを理解してください。

### 3. 発展的な学習
解答例を基に、さらなる改良や発展を考えてみてください。

### 4. デバッグの参考
自分の実装で問題が発生した場合の参考として活用してください。

## 重要な注意事項

### 学習の姿勢
- 解答例をただ写すのではなく、理解することを重視してください
- 自分なりの改良や変更を加えることを推奨します
- 異なる解法を考えて実装することが最も重要です

### 創造性の重要性
- 特に創作課題では、解答例は参考程度に留めてください
- 自分の感性や表現を大切にしてください
- オリジナリティを追求することが学習の目的です

### 技術的な理解
- 各行のコードの意味を理解してください
- パラメータを変更して効果を確認してください
- 数学的な背景も併せて学習してください

## 発展学習のヒント

### 1. パラメータの調整
各解答例のパラメータを変更して、異なる効果を体験してみてください。

### 2. 要素の組み合わせ
異なる解答例の要素を組み合わせて、新しい表現を作成してみてください。

### 3. 最適化の検討
より効率的な実装方法や、より美しい表現方法を考えてみてください。

### 4. 理論的な探求
使用されている数学的概念について、より深く学習してみてください。

## 質問やフィードバック

### よくある質問
1. **Q: 解答例と異なる実装をしたが、これで正しいか？**
   A: 正しく動作し、課題の要求を満たしていれば正解です。多様な解法を評価します。

2. **Q: 解答例より良い実装を思いついたが、どうしたらよいか？**
   A: 素晴らしいことです！ぜひ実装して、その改良点を記録してください。

3. **Q: 解答例の一部が理解できない**
   A: 段階的に理解を深めてください。Chapter 1の内容に戻って復習することも有効です。

### 学習コミュニティ
- 他の学習者と解答例を比較・議論することを推奨します
- 自分の実装について説明することで理解が深まります
- 異なる視点からの意見を積極的に求めてください
