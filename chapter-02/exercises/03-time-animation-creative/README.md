# 演習課題 3: 創作課題 - 時間とアニメーションの総合表現

## 目標
Chapter 2で学んだ時間とアニメーションの技術を総合的に活用して、オリジナルの視覚作品を創作する。

## 課題内容
これまで学んだ技術を自由に組み合わせて、以下のテーマから1つ以上を選んで創作してください：

### テーマ選択（複数選択可）

#### 3-1. 自然現象の表現
自然界の美しい現象をシェーダーで表現してください：

**推奨テーマ例:**
- **夕焼け・朝焼け**: 時間による色の変化で一日の移り変わりを表現
- **海の波**: 複数の波動を組み合わせた海面の表現
- **星空**: 点滅する星々と天の川の動き
- **花火**: 中心から放射状に広がる光の演出
- **雷**: 突然現れる電光の表現
- **雲の流れ**: 柔らかな雲の動きとその影

#### 3-2. 音楽・リズムの視覚化
音楽的なリズムやメロディを視覚的に表現してください：

**推奨アプローチ例:**
- **4/4拍子**: 規則的なリズムパターンの視覚化
- **メロディライン**: 音程の上下を波動で表現
- **ハーモニー**: 複数の周波数を重ね合わせた表現
- **楽器の特徴**: ピアノ、ギター、ドラムなどの音色を色で表現
- **音楽ジャンル**: クラシック、ジャズ、ロックなどの特徴を反映

#### 3-3. 抽象的な美しさ
数学的・幾何学的な美しさを追求してください：

**推奨アプローチ例:**
- **対称性**: 美しい対称パターンの創作
- **黄金比**: 1.618...を使った調和のとれた比率
- **フィボナッチ数列**: 自然界に現れる数列の視覚化
- **円周率**: π の無限性を表現
- **フラクタル風**: 自己相似性を持つパターン
- **色彩理論**: 補色、類似色を使った調和

#### 3-4. 感情・心理の表現
感情や心理状態を色と動きで表現してください：

**推奨表現例:**
- **静寂**: ゆっくりとした変化と落ち着いた色
- **躍動**: 活発な動きと鮮やかな色
- **神秘**: 深い色合いと予測不可能な動き
- **平和**: 穏やかな波動と暖かい色
- **緊張**: 急激な変化と対比の強い色
- **喜び**: 明るい色とリズミカルな動き

### 必須実装要素
作品には以下の要素を必ず含めてください：

1. **時間の活用**: `Time.duration` を使った動的な変化
2. **三角関数**: `sin` または `cos` を使った周期的な効果
3. **色の遷移**: 時間に応じた色の変化
4. **UV座標**: 位置に応じた効果の変化
5. **オリジナリティ**: 独自の工夫や表現

### 発展要素（任意）
より高度な表現を目指す場合、以下の要素も挑戦してください：

1. **複数の波動合成**: 3つ以上の異なる波動の組み合わせ
2. **非線形変化**: 二次関数、三次関数的な変化
3. **条件分岐**: 特定の条件での異なる表現
4. **色空間の活用**: HSV風の色作成
5. **数学的複雑性**: 複雑な数式の美しさ

1. **コンセプト決定**: 何を表現したいかを明確にする

3. **段階的構築**: 単純な要素から複雑な表現へ
4. **調整・改良**: パラメータを調整して理想的な表現に近づける
5. **完成度向上**: 細かい調整で作品の完成度を高める

作品制作において、以下を意識してください：
