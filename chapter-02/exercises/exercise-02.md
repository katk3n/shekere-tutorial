# 演習課題 2: sin/cosを使った様々なパターン作成

## 目標
sin/cos三角関数を使って、UV座標と時間を組み合わせた様々な視覚パターンを作成できるようになる。

## 課題内容
`examples/02-sin-cos-animation`をベースにして、以下の課題を順番に実装してください：

### 2-1. 基本的な波動パターン
UV座標を使った基本的な波動パターンを作成してください：

1. **水平波動**
   - 左から右へ移動する波のパターン
   - ヒント: `sin(uv.x * 周波数 + time * 速度)`

2. **垂直波動**
   - 上から下へ移動する波のパターン
   - ヒント: `sin(uv.y * 周波数 + time * 速度)`

3. **対角波動**
   - 斜めに移動する波のパターン
   - ヒント: `sin((uv.x + uv.y) * 周波数 + time * 速度)`

### 2-2. 円形・放射状パターン
中心点を基準とした円形や放射状のパターンを作成してください：

1. **同心円パターン**
   - 中心から広がる円形の波動
   - ヒント: `sin(length(uv) * 周波数 + time * 速度)`

2. **放射状パターン**
   - 中心から放射状に広がる波動
   - ヒント: `sin(atan2(uv.y, uv.x) * 周波数 + time * 速度)`

3. **螺旋パターン**
   - 螺旋状に回転する波動
   - ヒント: `sin(length(uv) * 周波数 + atan2(uv.y, uv.x) * 回転係数 + time * 速度)`

### 2-3. 複合パターン
複数の波動を組み合わせて、より複雑で美しいパターンを作成してください：

1. **十字パターン**
   - 水平と垂直の波動を組み合わせ
   - ヒント: `sin(uv.x * 周波数 + time) * sin(uv.y * 周波数 + time)`

2. **格子パターン**
   - 規則的な格子状のパターン
   - ヒント: `sin(uv.x * 周波数) + sin(uv.y * 周波数)`

3. **モアレパターン**
   - 異なる周波数を重ね合わせたパターン
   - ヒント: 2つの異なる周波数のsin波を加算

### 2-4. 創作パターン
自由な発想で独自のパターンを作成してください：

1. **自然現象の模倣**
   - 水面の波紋、雲の動きなど
   - 複数の波動を組み合わせて自然らしい動きを表現

2. **幾何学的美しさ**
   - 対称性や規則性を活かした美しいパターン
   - 数学的な美しさを視覚化

3. **オリジナル表現**
   - 自分だけのユニークなパターン
   - 色の変化も工夫して総合的な美しさを追求

### 基本構造
```wgsl
@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let uv = NormalizedCoords(in.position.xy);
    let time = Time.duration;
    
    // パターンの計算
    let pattern = sin(/* パターンの式 */);
    
    // 色の生成
    let intensity = (pattern + 1.0) * 0.5;
    let color = vec3(intensity, intensity * 0.8, intensity * 0.6);
    
    return vec4(ToLinearRgb(color), 1.0);

```

### 役立つ数学関数
- `length(uv)`: 中心からの距離
- `atan2(uv.y, uv.x)`: 中心からの角度
- `abs(x)`: 絶対値
- `fract(x)`: 小数部分
- `step(edge, x)`: 段階関数

### パターンの組み合わせ方法
```wgsl
// 加算（明るさを重ねる）
let combined = pattern1 + pattern2;

// 乗算（パターンを重ねる）
let combined = pattern1 * pattern2;

// 混合（段階的に合成）
let combined = mix(pattern1, pattern2, factor);

// 最大値（より明るい方を選択）
let combined = max(pattern1, pattern2);
```

### 色の創作テクニック
