### Step 4-1: 時間による自動アニメーション
**目標**: 時間軸を導入して自動的な美しい変化を作成

```wgsl
// 【新規追加】時間による回転と脈動
let time = Time.duration;

// 星座の回転
fn rotate_constellation(base_pos: vec2<f32>, time: f32, speed: f32) -> vec2<f32> {
    let angle = time * speed;
    let cos_a = cos(angle);
    let sin_a = sin(angle);
    return vec2(
        base_pos.x * cos_a - base_pos.y * sin_a,
        base_pos.x * sin_a + base_pos.y * cos_a
    );
}

// 光の脈動
let pulse = sin(time * 2.0) * 0.3 + 0.7;  // 0.4〜1.0の範囲で脈動
```

**学習ポイント**:
- `Time.duration`による時間の取得
- 三角関数による回転変換
- 周期的な変化による自然な脈動

### Step 4-2: パーティクル効果
**目標**: 動的な光の粒子による豊かな表現

```wgsl
// 【新規追加】パーティクル効果
fn particle_field(uv: vec2<f32>, mouse: vec2<f32>, time: f32) -> f32 {
    let particle_count = 20.0;
    var intensity = 0.0;
    
    for (var i = 0.0; i < particle_count; i += 1.0) {
        let angle = (i / particle_count) * 6.28318 + time * 0.5;
        let distance = 0.3 + 0.2 * sin(time * 1.5 + i);
        let particle_pos = mouse + vec2(cos(angle), sin(angle)) * distance;
        
        let particle_size = 0.02 + 0.01 * sin(time * 3.0 + i * 2.0);
        let particle_distance = length(uv - particle_pos);
        intensity += 1.0 / (1.0 + particle_distance * particle_distance / (particle_size * particle_size));
    }
    
    return intensity * 0.1;
}
```

**学習ポイント**:
- forループによる繰り返し処理
- 個別パーティクルの位置計算
- 時間による動的な位置・サイズ変化

### Step 4-3: エネルギー波動効果
**目標**: マウス位置から放射される波動の表現

```wgsl
// 【新規追加】エネルギー波動
fn energy_waves(uv: vec2<f32>, center: vec2<f32>, time: f32) -> f32 {
    let distance = length(uv - center);
    let wave_count = 3.0;
    var wave_intensity = 0.0;
    
    for (var w = 0.0; w < wave_count; w += 1.0) {
        let wave_speed = 2.0;
        let wave_frequency = 8.0;
        let wave_phase = time * wave_speed - w * 1.0;
        let wave = sin(distance * wave_frequency - wave_phase);
        let falloff = 1.0 / (1.0 + distance * 1.5);
        wave_intensity += max(0.0, wave) * falloff * 0.3;
    }
    
    return wave_intensity;
}
```

**学習ポイント**:
- 波動方程式の実装
- 複数の波の重ね合わせ
- 距離による減衰効果
