# 演習課題 1: 基本的なマウス追従効果

## 目標
`MouseCoords()`関数を使用して、マウス入力に応じた基本的なインタラクティブ効果を作成できるようになる。

## 課題内容
`examples/01-mouse-position`をベースにして、以下の効果を作成してください：

### 1-1. マウス位置による単一色変化
以下の効果を順番に作成し、それぞれの結果を確認してください：

1. **X座標のみで色変化**
   - マウスのX座標のみを赤成分に使用
   - 緑と青成分は固定値にする
   
2. **Y座標のみで色変化**
   - マウスのY座標のみを緑成分に使用
   - 赤と青成分は固定値にする

3. **反転効果**
   - マウスのX座標を使って青成分を制御
   - 左側で青が強く、右側で青が弱くなるように調整

### 1-2. 簡単な距離効果
以下の効果を作成してください：

1. **中心からの距離による明度変化**
   - 画面中央を基準とした距離を計算
   - 距離が近いほど明るく、遠いほど暗くなる効果

2. **マウスからの距離による色変化**
   - マウス位置からの距離を計算
   - 距離に応じて色を変化させる

### 1-3. 創作課題
以下の中から1つ選んで実装してください：

1. **虹色効果**
   - マウスのX座標に応じて色相が変化する効果
   
2. **スポットライト効果**
   - マウス位置を中心とした円形のスポットライト
   - 範囲外は暗くなる効果

## 実践方法
以下の形式でコードを書いて確認してください：

```wgsl
// 1-1-1. X座標のみで色変化
@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let mouse = MouseCoords();
    let red = (mouse.x + 1.0) * 0.5;
    let color = vec3(red, 0.3, 0.3);  // 緑と青は固定
    return vec4(ToLinearRgb(color), 1.0);
}

// 1-2-1. 中心からの距離による明度変化
@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let uv = NormalizedCoords(in.position.xy);
    let distance = length(uv);  // 中心からの距離
    // ここに距離を使った色の計算を追加
}
```

## 学習のポイント
- `MouseCoords()`は-1.0〜1.0の範囲で値を返します
- 色の成分は0.0〜1.0の範囲で指定する必要があります
- `(mouse.x + 1.0) * 0.5`で-1.0〜1.0を0.0〜1.0に変換できます
- Hot Reload機能を活用して、リアルタイムで変化を観察しましょう
- マウスを動かして効果を確認する習慣をつけましょう
