## 実装内容

### Step 3-1: 動的色相システム
**目標**: マウス位置に基づく色相の動的変化を実装

```wgsl
// 【新規追加】HSV色空間による色相計算
fn hsv_to_rgb(h: f32, s: f32, v: f32) -> vec3<f32> {
    let c = v * s;
    let x = c * (1.0 - abs(fract(h / 60.0) * 2.0 - 1.0));
    let m = v - c;
    
    var rgb = vec3<f32>(0.0);
    if (h >= 0.0 && h < 60.0) { rgb = vec3(c, x, 0.0); }
    else if (h >= 60.0 && h < 120.0) { rgb = vec3(x, c, 0.0); }
    else if (h >= 120.0 && h < 180.0) { rgb = vec3(0.0, c, x); }
    else if (h >= 180.0 && h < 240.0) { rgb = vec3(0.0, x, c); }
    else if (h >= 240.0 && h < 300.0) { rgb = vec3(x, 0.0, c); }
    else if (h >= 300.0 && h < 360.0) { rgb = vec3(c, 0.0, x); }
    
    return rgb + vec3(m);
}

// マウス位置による動的色相
let mouse_hue = (mouse.x + 1.0) * 180.0;  // -1〜1 を 0〜360度に変換
let mouse_color = hsv_to_rgb(mouse_hue, 0.8, 1.0);
```

**学習ポイント**:
- HSV色空間による直感的な色制御
- マウス座標から色相への変換
- 条件分岐による色相の段階的変換

### Step 3-2: 距離ベース色彩相互作用
**目標**: マウスと固定光源の距離による色彩変化

```wgsl
// 【新規追加】距離による色彩相互作用
fn get_interactive_color(base_hue: f32, mouse_pos: vec2<f32>, star_pos: vec2<f32>) -> vec3<f32> {
    let distance = length(mouse_pos - star_pos);
    let influence = 1.0 - clamp(distance / 2.0, 0.0, 1.0);
    
    // マウス位置の色相と固定光源の基本色相をブレンド
    let mouse_hue = (mouse_pos.x + 1.0) * 180.0;
    let blended_hue = mix(base_hue, mouse_hue, influence * 0.6);
    
    return hsv_to_rgb(blended_hue, 0.9, 1.0);
}
```

**学習ポイント**:
- 距離による影響度の計算
- 色相のブレンド（mix関数）
- 相互作用の強度制御

## 技術仕様

### 色彩パラメータ
- **主要光源の彩度**: 0.8〜1.0（鮮やかな色）
- **固定光源の彩度**: 0.6〜0.9（やや控えめ）
- **色相の影響範囲**: 距離2.0以内で最大影響
- **色相ブレンド強度**: 0.6（60%のブレンド）

### HSV色空間の活用
- **色相（Hue）**: 0〜360度、マウス位置で制御
- **彩度（Saturation）**: 0.6〜1.0、光源の種類で調整
- **明度（Value）**: 距離による減衰で制御

## 色彩デザインの考慮事項

### 美しい色彩関係
1. **補色関係**: 対比の美しさ（赤←→青緑）
2. **類似色関係**: 調和の美しさ（青→紫→赤紫）
3. **三色調和**: バランスの取れた組み合わせ

### 視覚的効果
- **グラデーション**: 滑らかな色の遷移
- **コントラスト**: 明暗・色相の対比
- **統一感**: 全体的な色彩の調和
