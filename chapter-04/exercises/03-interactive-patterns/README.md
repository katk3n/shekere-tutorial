# 演習課題 3: オリジナルインタラクティブパターン

## 目標
これまでに学んだ技術を組み合わせて、独自のインタラクティブシェーダーアートを作成する。

## 課題内容
以下のテーマの中から1つ選んで、オリジナルのインタラクティブパターンを作成してください：

### テーマ選択

#### A. 自然現象をモチーフにした効果
以下の自然現象をシェーダーで表現してください：

1. **星座**
   - マウス位置に星を配置
   - 周囲に小さな星を散らす
   - 星の明滅効果

2. **水面の波紋**
   - マウス位置を中心とした波紋
   - 複数の波紋が重なる効果
   - 時間による波の伝播

3. **オーロラ**
   - マウス位置に応じて色が変化
   - 流れるような色の変化
   - 複数色のグラデーション

#### B. 抽象的なアート表現
以下の抽象的表現を作成してください：

1. **万華鏡**
   - マウス位置を中心とした対称パターン
   - 回転する幾何学模様
   - 色の変化による動的効果

2. **エネルギー場**
   - マウス位置からのエネルギー放射
   - 力場の可視化
   - 動的な強度変化

3. **音楽的可視化**
   - マウス位置による音符の表現
   - リズミカルな色の変化
   - 楽器を模した効果

#### C. ゲーム的インタラクション
以下のゲーム要素を実装してください：

1. **魔法の杖**
   - マウス軌跡に沿った魔法効果
   - 粒子の軌跡
   - 色の変化と消失

2. **光の剣**
   - マウス位置からの光線
   - 鋭い光の効果
   - 色の強弱変化

3. **エネルギーシールド**
   - マウス位置を中心とした防御効果
   - 六角形や円形のシールド
   - 強度による色変化

## 実装のヒント

### 基本的な構造
```wgsl
// 独自の効果関数を定義
fn myEffect(uv: vec2<f32>, center: vec2<f32>, time: f32) -> vec3<f32> {
    // ここに効果の計算を記述
    return vec3(0.0, 0.0, 0.0);
}

@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let uv = NormalizedCoords(in.position.xy);
    let mouse = MouseCoords();
    let time = Time.duration;
    
    // 複数の効果を組み合わせ
    var col = vec3(0.0, 0.0, 0.0);
    col += myEffect(uv, mouse, time);
    
    return vec4(ToLinearRgb(col), 1.0);
}
```
