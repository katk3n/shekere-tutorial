## 実装内容

### Step 1-1: 基本的なマウス追従光源
**目標**: マウス位置を中心とした円形の光を作成

```wgsl
// マウス位置からの距離による光の強度計算
let uv = NormalizedCoords(in.position.xy);
let mouse = MouseCoords();
let distance = length(uv - mouse);
let intensity = 1.0 - clamp(distance / radius, 0.0, 1.0);
```

**学習ポイント**:
- `MouseCoords()`: -1.0〜1.0の範囲でマウス座標を取得
- `length(uv - mouse)`: マウス位置からの距離を計算
- `clamp()`: 値を指定範囲内に制限

### Step 1-2: 滑らかな光のフォールオフ
**目標**: より美しい光の減衰効果を作成

```wgsl
// 指数関数による滑らかな減衰
let smoothIntensity = pow(1.0 - clamp(distance / radius, 0.0, 1.0), 2.0);
```

**学習ポイント**:
- `pow()`: 指数関数による曲線的な変化
- 指数を変えることで減衰の急峻さを調整
- 自然な光の表現により近づける

### Step 1-3: 光源の色調整
**目標**: 光に適切な色彩を与える

```wgsl
// 暖かい光の色彩
let lightColor = vec3(1.0, 0.9, 0.7);  // 暖色系の光
let finalColor = lightColor * smoothIntensity;
```

**学習ポイント**:
- RGB値による色の表現
- 光らしい色彩の選択方法
- 強度との乗算による光の表現

## 技術仕様

### 推奨パラメータ
- **光の半径**: 0.3〜0.5
- **減衰指数**: 2.0〜4.0
- **光の色**: 暖色系 (1.0, 0.9, 0.7) または白色 (1.0, 1.0, 1.0)

### パフォーマンス考慮
- 距離計算は1回のみ実行
- 複雑な計算は避けて軽量に保つ
- 後続演習での拡張を考慮した構造
