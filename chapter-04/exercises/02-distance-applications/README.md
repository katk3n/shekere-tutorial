# 演習課題 2: 距離計算の応用

## 目標
距離計算を使用して、より高度なマウス追従効果を作成できるようになる。

## 課題内容
`examples/02-cursor-following`をベースにして、以下の効果を作成してください：

### 2-1. オーブ効果の改良
以下の改良を順番に実装してください：

1. **色の変更**
   - 緑色のオーブを赤色に変更
   - 青色のオーブを作成
   - 黄色のオーブを作成

2. **サイズの変更**
   - オーブの半径を0.1に変更
   - 半径を0.05に変更
   - 自分の好みのサイズを見つける

3. **フォールオフの調整**
   - `pow(t, 16.0)`の指数を変更して効果を観察
   - 4.0、8.0、32.0で試してみる

### 2-2. 複数のオーブ効果
以下の効果を作成してください：

1. **固定位置のオーブ**
   - 画面の四隅に固定のオーブを配置
   - それぞれ異なる色にする

2. **マウス追従 + 固定オーブ**
   - マウスに追従するオーブ1つ
   - 画面中央に固定のオーブ1つ
   - 両方を同時に表示

3. **マウス周辺のオーブ**
   - メインオーブ（マウス位置）
   - サブオーブ（マウス位置から少しずらした位置）
   - 合計3つのオーブを表示

### 2-3. 高度な効果
以下の中から1つ選んで実装してください：

1. **グラデーション効果**
   - マウス位置から遠ざかるほど色が変化
   - 滑らかなグラデーションを作成

2. **パルス効果**
   - 一定間隔でオーブが拡大・縮小
   - ヒント: `sin(Time.duration * 速度)`を使用

3. **軌跡効果**
   - マウスの軌跡に沿って複数のオーブを表示
   - 時間差で色が変化

## 実践方法
以下の形式でコードを書いて確認してください：

```wgsl
// 2-1-1. 色の変更（赤色のオーブ）
fn orb(p: vec2<f32>, p0: vec2<f32>, r: f32, col: vec3<f32>) -> vec3<f32> {
    let t = clamp(1.0 + r - length(p - p0), 0.0, 1.0);
    return vec3(pow(t, 16.0) * col);
}

@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let uv = NormalizedCoords(in.position.xy);
    let mouse = MouseCoords();
    
    let red = vec3(1.0, 0.0, 0.0);  // 赤色に変更
    var col = vec3(0.0, 0.0, 0.0);
    col += orb(uv, mouse, 0.07, red);
    
    return vec4(ToLinearRgb(col), 1.0);
}

// 2-2-1. 固定位置のオーブ
@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let uv = NormalizedCoords(in.position.xy);
    
    // 四隅の固定位置
    let corner1 = vec2(-0.8, -0.8);
    let corner2 = vec2(0.8, -0.8);
    let corner3 = vec2(-0.8, 0.8);
    let corner4 = vec2(0.8, 0.8);
    
    // 異なる色のオーブを配置
    var col = vec3(0.0, 0.0, 0.0);
    col += orb(uv, corner1, 0.05, vec3(1.0, 0.0, 0.0));  // 赤
    col += orb(uv, corner2, 0.05, vec3(0.0, 1.0, 0.0));  // 緑
    col += orb(uv, corner3, 0.05, vec3(0.0, 0.0, 1.0));  // 青
    col += orb(uv, corner4, 0.05, vec3(1.0, 1.0, 0.0));  // 黄
    
    return vec4(ToLinearRgb(col), 1.0);
}
```

## 学習のポイント
- `clamp()`関数で値を0.0〜1.0の範囲に制限する
- `pow()`関数の指数を変更することで効果の強さを調整
- 複数のオーブを組み合わせる時は色の加算（`col += orb(...)`）を使用
- `Time.duration`を使って時間による変化を追加できる
- 数値を少しずつ変更して、自分好みの効果を見つけましょう
