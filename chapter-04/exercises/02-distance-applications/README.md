## 実装内容

### Step 2-1: 光源関数の作成
**目標**: 再利用可能な光源生成関数を作成

```wgsl
// 【新規追加】汎用光源関数
fn create_light(uv: vec2<f32>, center: vec2<f32>, radius: f32, 
               intensity: f32, color: vec3<f32>) -> vec3<f32> {
    let distance = length(uv - center);
    let normalized_distance = clamp(distance / radius, 0.0, 1.0);
    let light_strength = pow(1.0 - normalized_distance, 3.0) * intensity;
    return color * light_strength;
}
```

**学習ポイント**:
- 関数による処理の再利用
- パラメータによる光源の個性化
- 戻り値による色の計算

### Step 2-2: 固定光源の配置
**目標**: 画面上の戦略的位置に美しい光源を配置

```wgsl
// 【新規追加】固定位置の星座光源
let star1 = create_light(uv, vec2(-0.6, 0.4), 0.2, 0.7, vec3(0.8, 0.9, 1.0));
let star2 = create_light(uv, vec2(0.5, -0.3), 0.15, 0.6, vec3(1.0, 0.8, 0.6));
let star3 = create_light(uv, vec2(0.0, 0.6), 0.25, 0.5, vec3(0.9, 0.7, 1.0));
```

**学習ポイント**:
- 固定座標による配置
- 異なるサイズ・強度・色の光源
- 視覚的バランスの考慮

### Step 2-3: 光源の合成
**目標**: 複数の光源を美しく組み合わせる

```wgsl
// 【継承+新規追加】全光源の合成
var total_light = vec3(0.0, 0.0, 0.0);

// 前ステップから継承: マウス追従メイン光源

// 新規追加: 固定星座光源

```

**学習ポイント**:
- 加算による光の合成

### 光の階層構造
