# 演習課題 3: 創作課題

## 目標
Chapter 3で学んだ技術を総合的に活用し、独自の芸術的なパターンを作成する。

## 課題内容
これまでに学んだ座標変換、距離関数、角度関数、そして時間アニメーションを組み合わせて、独自のシェーダーアートを作成してください。

### 3-1. テーマ選択
以下のテーマから一つを選んで作品を作成してください：

1. **自然界のパターン**
   - 例: 花、雪の結晶、木の年輪、波紋、稲妻など
   - 自然界に存在するパターンをシェーダーで再現

2. **幾何学的パターン**
   - 例: 万華鏡、アラベスク、フラクタル、タイリングなど
   - 数学的な美しさを表現

3. **抽象的なパターン**
   - 例: 感情の表現、音楽の視覚化、エネルギーの流れなど
   - 抽象的な概念を視覚的に表現

4. **未来的なパターン**
   - 例: ホログラム、エネルギーフィールド、デジタル空間など
   - SF的な要素を取り入れたパターン

### 3-2. 必須要素
作品には以下の要素を必ず含めてください：

1. **座標変換**
   - スケーリング、移動、回転、対称化のいずれか

2. **距離関数の活用**
   - `length()`を使った円形パターン

3. **角度関数の活用**
   - `atan2()`を使った放射状パターン

4. **時間アニメーション**
   - `Time.duration`を使った動的な変化

5. **色の変化**
   - 位置や時間による色の変化

### 3-3. 発展要素（任意）
余裕があれば以下の要素も取り入れてください：

1. **複数パターンの合成**
   - 異なるパターンを組み合わせる

2. **非線形変換**
   - `pow()`, `exp()`, `log()`などによる変形

3. **ノイズ風効果**
   - 疑似乱数による自然な変化

4. **インタラクティブ要素**
   - マウス座標の活用（次章の先取り）

    let angle = atan2(uv.y, uv.x);

    // 色の計算

    return vec4(ToLinearRgb(color), 1.0);
}

## 評価のポイント
- 技術的な正確性
- 創造性と独自性
- 美しさと芸術性
- 学習内容の理解度
- コードの読みやすさ

## 学習のポイント
- 完璧を目指さず、まずは動くものを作りましょう
- 小さな変更を積み重ねて改良していきましょう
- 他の人の作品を参考にしても構いません
- 実験を恐れず、様々なパラメータを試してみましょう
- 自分なりの表現を見つけることが重要です
