# 演習課題 2: 数学的パターンの創作

## 目標
距離関数と角度関数を使って、独自の数学的パターンを作成する。

## 課題内容
`examples/02-circular-patterns`と`examples/03-radial-patterns`をベースにして、以下のパターンを作成してください：

### 2-1. 距離関数の応用
距離関数を使って以下のパターンを作成してください：

1. **パルス円**
   - 時間によって半径が変化する円
   - ヒント: `step(distance, sin(time) * 0.3 + 0.5)`

2. **多重同心円**
   - 3つの異なる中心から同心円を描く
   - ヒント: 複数の`length(uv - center)`を計算

3. **波動円**
   - 円の境界が波動する円
   - ヒント: `sin(atan2(uv.y, uv.x) * 波数)`を半径に加算

### 2-2. 角度関数の応用
角度関数を使って以下のパターンを作成してください：

1. **花びらパターン**
   - 花びらのような形状
   - ヒント: `sin(angle * 花びら数) * 振幅`を半径に適用

2. **螺旋パターン**
   - 螺旋状のパターン
   - ヒント: `angle + distance * 係数`で螺旋を作成

3. **分割円**
   - 円を扇形に分割
   - ヒント: `fract(angle * 分割数)`で分割

### 2-3. 距離と角度の組み合わせ
両方の関数を組み合わせて以下のパターンを作成してください：

1. **極座標グリッド**
   - 放射状の格子パターン
   - 同心円と放射線の組み合わせ

2. **ダーツボード**
   - ダーツボードのような分割パターン
   - 距離による同心円と角度による分割

3. **万華鏡**
   - 対称性を活用した万華鏡パターン
   - 複数の対称軸を持つパターン

## 実践方法
以下の形式でコードを書いて確認してください：

```wgsl
@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let uv = NormalizedCoords(in.position.xy);
    let time = Time.duration;
    
    // 2-1-1. パルス円
    let distance = length(uv);
    let pulse_radius = sin(time * 2.0) * 0.3 + 0.5;
    let pulse_circle = step(distance, pulse_radius);
    
    // 色の計算
    let color = vec3(pulse_circle, pulse_circle * 0.8, pulse_circle * 0.6);
    
    return vec4(ToLinearRgb(color), 1.0);
}
```

## 発展課題
余裕があれば以下にもチャレンジしてください：

1. **アニメーション化**
   - 時間と組み合わせて動きを追加

2. **色の変化**
   - パターンに応じた色の変化

3. **複数パターンの合成**
   - 異なるパターンを重ね合わせる

4. **独自パターンの創作**
   - 完全にオリジナルなパターンを作成

## 学習のポイント
- 数学関数の組み合わせで無限の可能性があります
- 小さな変化でも大きく見た目が変わります
- 対称性を活用すると美しいパターンが作れます
- 実際に数値を変えて実験することが重要です
- 自然界のパターンを参考にすると良いアイデアが浮かびます
