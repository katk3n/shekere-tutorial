# 演習課題 1: 座標変換の応用

## 目標
UV座標の理解を深め、基本的な座標変換テクニックを習得する。

## 課題内容
`examples/01-uv-coordinates`をベースにして、以下の座標変換を実装してください：

### 1-1. 座標の対称化
以下の対称パターンを作成してください：

1. **水平対称**
   - ヒント: `abs(uv.x)`を使用
   - 左右が対称になるパターン

2. **垂直対称**
   - ヒント: `abs(uv.y)`を使用  
   - 上下が対称になるパターン

3. **中心対称**
   - ヒント: `abs(uv)`を使用
   - 4つの象限が同じパターン

### 1-2. 座標のスケーリング
以下のスケーリングパターンを作成してください：

1. **2倍拡大**
   - 座標を2倍にスケーリング
   - ヒント: `uv * 2.0`

2. **0.5倍縮小**
   - 座標を半分にスケーリング
   - ヒント: `uv * 0.5`

3. **非等方スケーリング**
   - X方向に2倍、Y方向に0.5倍
   - ヒント: `vec2(uv.x * 2.0, uv.y * 0.5)`

### 1-3. 座標の移動
以下の移動パターンを作成してください：

1. **右に移動**
   - 座標を右に0.5移動
   - ヒント: `uv + vec2(0.5, 0.0)`

2. **左上に移動**
   - 座標を左に0.3、上に0.3移動

3. **円形移動**
   - 時間による円形移動
   - ヒント: `uv + vec2(cos(time), sin(time)) * 0.3`

## 実践方法
以下の形式でコードを書いて確認してください：

```wgsl
@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    // 基本座標
    let uv = NormalizedCoords(in.position.xy);
    
    // 1-1-1. 水平対称
    let transformed_uv = vec2(abs(uv.x), uv.y);
    
    // 色の計算
    let color = vec3(
        (transformed_uv.x + 1.0) * 0.5,
        (transformed_uv.y + 1.0) * 0.5,
        0.3
    );
    
    return vec4(ToLinearRgb(color), 1.0);
}
```

## 発展課題
余裕があれば以下にもチャレンジしてください：

1. **座標の繰り返し**
   - `fract(uv * 3.0)`でタイル状に繰り返し

2. **座標の回転**
   - 座標を45度回転させる

3. **複合変換**
   - 複数の変換を組み合わせる
