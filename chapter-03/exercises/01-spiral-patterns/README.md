# 演習 1: 螺旋パターン「銀河の腕 - Galaxy Arms」

## 目標
length/atan2関数を使って銀河の渦巻き構造を数学的に再現し、Chapter 3の核心技術である極座標系（距離・角度）による表現方法を習得する。

## 学習内容
- **distance関数**: `length()`による中心からの距離計算
- **angle関数**: `atan2()`による角度計算と正規化
- **極座標系**: 距離と角度による座標表現の基礎
- **螺旋パターン**: 角度と距離を組み合わせた螺旋形状の生成
- **fract関数**: 周期化による繰り返しパターンの作成

## 作品テーマ: 「自然界の数学的パターンの発見」シリーズ第1作
この演習では、「自然の複合パターン - Natural Complex Patterns」作品への第1ステップとして、銀河の美しい渦巻き構造を数学的に再現します。宇宙に存在する螺旋銀河の腕の構造を、シンプルな数学関数で表現する方法を学びます。

## 技術要素
### 新規実装技術（この演習で学ぶ内容）
- **`length(uv)`**: 中心からの距離計算（0〜約1.4の範囲）
- **`atan2(uv.y, uv.x)`**: 角度計算（-π〜π範囲）
- **角度の正規化**: `(angle + PI) / (2.0 * PI)` で0〜1の扱いやすい範囲に変換
- **`fract(angle + distance)`**: 螺旋パターン生成の数学的手法
- **極座標表現**: デカルト座標系(x,y)から極座標系(r,θ)への変換

## 段階的実装手順

このexerciseでは、以下の6つのステップを順番に実装することで、螺旋パターンを段階的に理解し構築します。

### Step 1: 基本座標取得（環境確認）

**目標**: shekere環境の動作確認と基本座標の取得

**新規追加コード**:
```wgsl
const PI = 3.14159265359;

let uv = NormalizedCoords(in.position.xy);
let color = vec3(0.5, 0.3, 0.8);  // 単色で環境確認
```

**解説**: `NormalizedCoords()`で正規化座標(-1〜1の範囲)を取得し、単色で表示します。

**確認方法**: 画面全体が紫色で表示されることを確認してください。

---

### Step 2: 距離計算の追加（中心からの距離で色変化）

**目標**: `length()`関数を使って中心からの距離を計算し、距離に応じて色を変化させる

**新規追加コード**:
```wgsl
// 新規追加: 中心からの距離を計算
let distance = length(uv);

// 変更: 距離に応じて色を変化させる
let color = vec3(distance, distance * 0.5, distance);
```

**解説**: `length(uv)`で中心(0,0)からの距離を計算します。中心が暗く、外側に向かって明るくなるグラデーションが生成されます。

**確認方法**: 中心が黒く、外側に向かって紫がかったグラデーションが表示されることを確認してください。

---

### Step 3: 角度計算の追加（角度で色変化を確認）

**目標**: `atan2()`関数を使って角度を計算し、角度による色の変化を確認する

**新規追加コード**:
```wgsl
// 新規追加: 角度の計算
let angle = atan2(uv.y, uv.x);

// 新規追加: 角度を色で表示（-π〜πを0〜1に変換）
let angle_color = (angle + PI) / (2.0 * PI);
let color = vec3(angle_color, distance * 0.3, angle_color * 0.8);
```

**解説**: `atan2(uv.y, uv.x)`で角度(-π〜π)を計算し、色で表示します。円周に沿って色が変化することで角度計算が確認できます。

**確認方法**: 中心から放射状に色が変化し、円周に沿って色が移り変わることを確認してください。

---

### Step 4: 角度の正規化（0〜1範囲への変換）

**目標**: 角度を0〜1の扱いやすい範囲に正規化する

**新規追加コード**:
```wgsl
// 新規追加: 角度を0〜1範囲に正規化
let normalized_angle = (angle + PI) / (2.0 * PI);

// 変更: 正規化された角度を表示
let color = vec3(normalized_angle, distance * 0.5, normalized_angle * distance);
```

**解説**: `(angle + PI) / (2.0 * PI)`で-π〜πの角度を0〜1の範囲に変換します。これにより後の計算が扱いやすくなります。

**確認方法**: Step 3と似ていますが、より均等で滑らかな色の変化が確認できるはずです。

---

### Step 5: 螺旋パターンの生成（核心技術）

**目標**: `fract()`関数を使って螺旋パターンを生成する

**新規追加コード**:
```wgsl
// 新規追加: 螺旋パターンの生成！
let spiral_frequency = 3.0;  // 螺旋の腕の数
let spiral_tightness = 5.0;  // 螺旋の巻き具合
let spiral_pattern = fract(normalized_angle * spiral_frequency + distance * spiral_tightness);

// 変更: 螺旋パターンを表示
let color = vec3(spiral_pattern, spiral_pattern, spiral_pattern);
```

**解説**: `fract(normalized_angle * frequency + distance * tightness)`が螺旋パターンの核心です。角度×頻度で腕の数、距離×巻き具合で螺旋の密度を制御します。

**確認方法**: 銀河のような美しい螺旋パターンが白黒で表示されることを確認してください。

---

### Step 6: 色彩の調整（最終完成）

**目標**: 銀河をイメージした美しい色彩に調整して完成

**変更コード**:
```wgsl
// 変更: 銀河をイメージした青〜紫系の色彩
let color = vec3(
    spiral_pattern * 0.6,         // 赤成分（紫のため中程度）
    spiral_pattern * 0.4,         // 緑成分（低め）
    spiral_pattern               // 青成分（強め）
);
```

**解説**: RGB成分を調整して銀河のような青〜紫系の美しい色彩を実現します。各成分の比率により色調が決まります。

**確認方法**: 美しい青紫色の螺旋銀河パターンが表示されることを確認してください。完成です！

### 技術解説

#### 1. 極座標系への変換
```wgsl
let distance = length(uv);      // 中心からの距離
let angle = atan2(uv.y, uv.x);  // 角度（-π〜π）
```
- `length()`: ベクトルの大きさ（原点からの距離）
- `atan2()`: 逆正接関数で角度を求める（4象限対応）

#### 2. 角度の正規化
```wgsl
let normalized_angle = (angle + PI) / (2.0 * PI);
```
- `-π〜π`の範囲を`0〜1`の扱いやすい範囲に変換
- 色や他の計算で使いやすくなる

#### 3. 螺旋パターンの生成
```wgsl
let spiral_pattern = fract(normalized_angle * spiral_frequency + distance * spiral_tightness);
```
- `normalized_angle * frequency`: 角度方向の繰り返し（腕の数）
- `distance * tightness`: 距離による螺旋の巻き具合
- `fract()`: 0〜1の範囲に周期化して螺旋パターンを作成
